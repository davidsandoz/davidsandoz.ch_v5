# [davidsandoz.ch](https://davidsandoz.ch)
Personal website made with [Angular](https://angular.io).

Based on [angular-seed](https://github.com/mgechev/angular-seed).

## Setup
```
$ npm install
$ npm start
```

## Deploy
```
$ npm run build.prod
```
And deploy the content of `dist/prod/` to a web server.
